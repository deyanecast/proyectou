Conceptos

Firmware


El firmware es un tipo de software que se
graba directamente en una pieza de hardware.

MiddleWare


El middleware es un software que se encuentra
entre un sistema operativo y
las aplicaciones que se ejecutan en él.

Cuantificación de bits y byte


Los bits son dígitos binarios, un bit
puede tener valor 0 o 1.
Los bytes están compuestos por 8 bits cada uno

Arquitectura de un Sistema Operativo


Es un modelo en el cual el sistema operativo y
todos los servicios fundamentales
residen en un monitor monolítico que se accede a
través de un mecanismo de llamada al núcleo.

Bases de Datos en MySQL


Una base de datos se define como un conjunto de datos organizados,
los cuales guardan una cierta relación entre sí.
Así, los sistemas de información se encargan de recolectarlos y, posteriormente, gestionarlos y administrarlos para unos objetivos concretos.
Las bases de datos tienen una serie de características que resulta interesante conocer.




Componentes de comunicación

Sistema de Bases numéricas


La base indica el número de cifras o dígitos que usa un
conjunto de símbolos y reglas que permiten
representar datos numéricos (llamado Sistema numérico)


Introducción a Redes


Una red es un conjunto de ordenadores conectados entre si, que pueden comunicarse para compartir datos y
recursos sin importar la localización física de los distintos dispositivos.


Implementación de Redes


Las conexiones seguras a Internet respaldan tu privacidad en el envío de datos y uno de los
procedimientos más simples para navegar con esta tranquilidad es
la utilización de certificados Secure Sockets Layer (SSL)


Manejo de octetos para la implementación de Redes


Una dirección IP es una dirección empleada para identificar a un dispositivo en una red IP.
La dirección se compone de 32 bits binarios, que pueden dividirse en
una porción correspondiente a la red y otra correspondiente al host, con la ayuda de una máscara de subred.




Resolución de Problemas y Pseudocódigo

Planteamiento y Resolución de Problemas.


El planteamiento de problemas puede ocurrir dentro del proceso de resolución de problemas,
cuando el resolutor, al resolver un problema no trivial, se implica de modo que se puede decir que
plantea un nuevo problema de alguna manera para
hacerlo más accesible y poderlo resolver


Algoritmos y Diagramas.


Los algoritmos y los diagramas de flujo son dos herramientas diferentes
que resultan útiles para crear nuevos programas,
especialmente en la programación informática.


Tipos de Operadores.


Tipos de Operadores.


Constantes, variables, tipos de instrucciones.


En los programas existirán conceptos que poseerán valores, estos conceptos reciben el nombre de variables.
Las variables poseen en un momento de la ejecución del algoritmo un único valor,
pero este valor puede ir cambiando a medida que se van ejecutando instrucciones, de allí el nombre de variable.
